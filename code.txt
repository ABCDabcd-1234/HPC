🧠 Matrix Multiplication Formula
For:
C = A × B
Each element:
C[i][j] = Σ A[i][k] × B[k][j]

📚 Example
Suppose:
A =1 23 4B =5 67 8
Then:
C[0][0]= (1×5) + (2×7)= 5 + 14= 19

🔍 This Line
C_seq[i][j] += A[i][k] * B[k][j];
Accumulates sum.

🧠 Complexity

N × N × N

🚀 PARALLEL MATRIX MULTIPLICATION

🌟 MOST IMPORTANT LINE
#pragma omp parallel for
This creates multiple threads.

🧠 What happens internally?
Without OpenMP:
1 CPU core executes all rows
With OpenMP:
Multiple CPU cores divide rows
Example:
ThreadRowsT10–199T2200–399T3400–599T4600–799

🌟 Why parallelism works here?
Each row computation is independent.
Thread computing:
C[0][*]
does NOT affect:
C[1][*]
So safe to parallelize.

🧵 THREAD EXECUTION
Each thread computes:
C_par[i][j]
independently.
No race conditions occur.

⚠️ Why no critical needed?
Because:
Each thread writes different memory locations
Example:
Thread 1 → C[0][0]Thread 2 → C[1][0]
No shared write conflicts.

⏱️ PARALLEL TIME
Measures OpenMP execution speed.

⚡ SPEEDUP

🧠 Viva One-Liners

What is OpenMP?

OpenMP is an API for shared-memory parallel programming in C/C++.


Why matrix multiplication important in AI/ML?

Neural networks perform massive matrix multiplications during forward and backward propagation.


Why parallelization works here?

Each row computation is independent, so threads can execute simultaneously.


Why no race condition?

Each thread writes to separate matrix elements.


What is speedup?

Ratio of sequential execution time to parallel execution time.
