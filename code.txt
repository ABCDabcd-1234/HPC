This notebook builds a Recurrent Neural Network (RNN) model to predict Google stock prices using previous stock prices.

I’ll explain every line, concept, and logic in very small details.

OVERALL GOAL OF THIS PROJECT

The notebook tries to answer:

“Can we predict tomorrow’s Google stock opening price using previous 60 days’ prices?”

This is a Time Series Forecasting problem because:

Data changes over time
Previous values influence future values

Example:

Day	Open Price
Day 1	100
Day 2	102
Day 3	101
Day 4	?

The model learns patterns from past prices.

WHAT IS RNN?

RNN = Recurrent Neural Network

It is a neural network specially designed for:

sequential data
time series
speech
text
stock prices

Why?

Because RNN remembers previous information.

Unlike normal neural networks:

Traditional NN → treats all inputs independently
RNN → remembers previous inputs

Example:

To predict today’s stock price:

yesterday’s price matters
last week’s trend matters
last 60 days matter

RNN captures this sequence dependency.
