🌟 CUDA EXECUTION MODEL
CUDA organizes execution into:
Grid → Blocks → Threads

🌟 THREAD
Smallest execution unit.
Each thread:
executes kernel code

🌟 BLOCK
Group of threads.
Example:
BLOCK_SIZE = 2
Matrix multiplication:
2x2 = 4 threads/block

🌟 GRID
Collection of blocks.
Entire GPU execution launched as:
grid of blocks

🌟 VECTOR ADDITION KERNEL

✅ Kernel Definition
__global__ void vectorAdd(...)

🌟 What is __global__?
Means:
function runs on GPU
Called from CPU.

🌟 KERNEL
GPU function.
Executed:
by thousands of threads

🌟 Thread Index Calculation
int i = blockIdx.x * blockDim.x + threadIdx.x;
MOST IMPORTANT CUDA FORMULA.

🌟 Special CUDA Variables

✅ blockIdx.x
Current block number.

✅ blockDim.x
Threads per block.

✅ threadIdx.x
Current thread inside block.

🌟 Example
Suppose:
BLOCK_SIZE = 2numBlocks = 3
Threads:
BlockThreadi000011102113204215
Each thread handles:
one vector element

if (i < N)
Needed because:

🌟 VECTOR ADDITION

🌟 HOST VS DEVICE MEMORY

✅ Host
Host =
CPU memory (RAM)
Variables:
hA hB hC

✅ Device
Device =
GPU memory
Variables:
dA dB dC

🌟 HOST ARRAY CREATION
int *hA = new int[N];
Allocates CPU memory.

🌟 GPU MEMORY ALLOCATION
cudaMalloc(&dA, size);
Allocates:
GPU memory
Equivalent of:
malloc()
but on GPU.

🌟 DATA TRANSFER
cudaMemcpy(dA, hA, size, cudaMemcpyHostToDevice);
Copies:
CPU → GPU

🌟 Why Needed?
GPU cannot directly access CPU memory.
Data must be copied.

🌟 Kernel Launch
vectorAdd<<<numBlocks, BLOCK_SIZE>>>(...)
CUDA launch syntax.

🌟 Meaning
<<< blocks, threads_per_block >>>
Example:
<<<3,2>>>
means:
3 blocks2 threads/block6 total threads

🌟 Number of Blocks
(N + BLOCK_SIZE - 1) / BLOCK_SIZE
Rounds up division.

🌟 Example
If:
N = 5BLOCK_SIZE = 2
Need:
3 blocks
Because:
2+2+1 elements

🌟 Copy Back Result
cudaMemcpy(hC, dC, size, cudaMemcpyDeviceToHost);
Copies:
GPU → CPU

🌟 MATRIX MULTIPLICATION

🌟 Goal
Compute:
C=A×BC = A \times BC=A×B

🌟 Matrix Multiplication Formula
Cij=∑k=0N−1AikBkjC_{ij} = \sum_{k=0}^{N-1} A_{ik}B_{kj}Cij​=∑k=0N−1​Aik​Bkj​

🌟 Example
If:
A=[1234]A =
\begin{bmatrix}
1 & 2 \\
3 & 4
\end{bmatrix}A=[13​24​]
B=[5678]B =
\begin{bmatrix}
5 & 6 \\
7 & 8
\end{bmatrix}B=[57​68​]
Then:
C00=1×5+2×7C_{00} = 1\times5 + 2\times7C00​=1×5+2×7

🌟 Matrix Kernel
__global__ void matrixMul(...)
Each thread computes:
one matrix cell

🌟 Row and Column Calculation
int row = blockIdx.y * blockDim.y + threadIdx.y;int col = blockIdx.x * blockDim.x + threadIdx.x;
Determines:
which matrix element this thread computes

🌟 Dot Product Loop
for (int n = 0; n < N; ++n)
Computes:
row-column multiplication sum

🌟 Example
For:
C[0][1]
Thread computes:
A[0][0]*B[0][1]+A[0][1]*B[1][1]+...
